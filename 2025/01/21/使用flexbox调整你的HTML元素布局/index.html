使用flexbox调整你的HTML元素布局

介绍

Flexbox 是一组用于对齐块级内容的 CSS 属性，包含两种内容：

  • Container：即在HTML中被直接控制的目标元素（flex parent）
  • Items：被嵌套在父元素中的元素，真正需要被控制位置的部分（flex children）
